## Featured Projects

### Predictive Battery Health Monitoring System

> **Tech Stack:** Python · LSTM (Deep Learning) · Time-Series Analysis · Anomaly Detection

Designed and implemented a sophisticated battery health monitoring system leveraging **Long Short-Term Memory (LSTM)** neural networks to predict **State-of-Health (SOH)** and **Remaining Useful Life (RUL)** for battery packs exhibiting cell imbalance. The system performs real-time estimation, enabling proactive maintenance strategies, early fault detection, and optimization of battery lifecycle management — directly applicable to EV, BMS, and renewable energy storage domains.

**Key Highlights:**
- Real-time SOH & RUL estimation with LSTM-based time-series modelling
- Cell imbalance detection and anomaly flagging for early intervention
- Supports proactive maintenance strategies to reduce downtime
- Applicable to EV batteries, UPS systems, and IoT energy devices

### Temple Crowd Tracking System (AI-Based Surveillance)

> **Tech Stack:** Python · OpenCV · YOLOv8 · Computer Vision · Data Logging

Built an intelligent, real-time crowd monitoring system for public spaces using **YOLOv8** object detection and **OpenCV** for video stream processing. The system performs live people counting, density estimation, and behavioral analytics — generating automated reports for smart surveillance and crowd management authorities.

**Key Highlights:**
- Real-time people detection and counting using YOLOv8
- Crowd density heatmaps and zone-wise analytics
- Automated data logging and report generation
- Configurable alert thresholds for overcrowding scenarios
